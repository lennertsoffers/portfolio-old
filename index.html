16-17 October 2021

Ypto Hackathon

PowerApps

Powerpoint

We had to come up with a solution that would allow us to inventorise the components of a wagon. This was done using Microsoft's PowerApps platform. What set our solution apart from others, is the idea that we would also keep track of the circumstances in which components break down. In this way, we can anticipate the problem by finding a solution for the cause instead of having to replace the component each time. With this solution we won the main prize of the hackathon, an exclusive tour of the Microsoft headquarters of Paris.

On this website you can find our pitch an the evaluation of the jury: www.ypto.be/hackathon
