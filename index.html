16-17 October 2021

Ypto Hackathon

PowerApps

Powerpoint

Microsoft and Ypto challenged us with the problem: How to inventorise the componenets of a wagon. Our solution was an inventorisation application with which you can scan each components its digital tag and than enter the status of the component (if it needs reparation, replacement...). What set our solution apart from others, is the idea that we would also keep track of the circumstances in which components break down. In this way, we can anticipate the problem by finding a solution for the cause instead of having to replace the component each time. With this solution we won the main prize of the hackathon, an exclusive tour of the Microsoft headquarters in Paris.

1 November 2021

ElementalStones

Java

Spigot API

Elemental Stones is the largest project I have worked on so far. It is a plugin for the game 'Minecraft' written in Java. The plugin increases the features that the user has in the game and works server-side. While at school we only saw the basics of Java, during this project I have learned a lot about the programming language and I now understand more complicated concepts. Documentation also plays an important role in the creation of plugins, so I focused on this during the project. I learned working with git and created a dedicated wiki page. I also experienced the power of testing and why other people should test your software too. The plugin now has over 150 downloads on Spigot.mc.

19 December 2021

PokePlayer

C#

WPF

Dapper

For the C# course, we had to create a WPF applicating using an API and the micro ORM Dapper. I took this assignment to the next level by challenging myself with building a game in WPF. PokePlayer was my solution, this is a game that offers the same functionality as the old Pokemon games on the Nintendo DS. For this application I used the PokeAPI which provides the data. This was my first experience with C# and WPF. In addition, I used GitHub workflows to accelerate the release of the application.
